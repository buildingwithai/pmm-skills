# PMM Skills

Product marketing skills for Claude Code agents.

## For Agents Creating New Skills

**⚠️ IMPORTANT:** Before creating a new skill, read [`SKILL_CREATION_GUIDE.md`](SKILL_CREATION_GUIDE.md)

This guide contains:
- Step-by-step skill creation process
- Best practices and limits (500 line SKILL.md, 8-12 reference files)
- How to build action-oriented, operational guides
- Quality checklist and anti-patterns to avoid

## Available Skills

### 1. Multi-Audience Product Messaging
**Path:** `multi-audience-product-messaging/`

Create positioning and messaging that resonates with both technical and non-technical audiences. Includes April Dunford's framework, Pain-Claim-Gain methodology, workshop facilitation, message testing, and sales stage strategies.

### 2. Personas & User Research
**Path:** `personas-user-research/`

Master creating, validating, and activating buyer and user personas. Covers research methodologies, interview techniques (5 Rings), validation methods, and organization-wide activation strategies.
